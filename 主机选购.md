# 低功耗迷你主机选购

目标：买一台 **日常办公 / 浏览器 / 网盘 / 影音** 用的低功耗、安静型 mini PC。

当前目标已经收敛为：**二线品牌 N100 / N150，16G+512G，目标价 800-1100 元**。

## 购买口径

| 项目       | 判断                                                                         |
| :--------- | :--------------------------------------------------------------------------- |
| CPU        | Intel N100 / N150 即可，日常办公差距不大；N150 略新一点。                    |
| 内存       | 优先 16G，8G 不建议作为主力办公机。                                          |
| 硬盘       | 优先 512G；128G 直接排除，256G 只在价格很低时考虑。                          |
| 品牌       | 二线品牌优先：极摩客、天虹、磐镭、天钡、摩方、MAXHUB、大唐 Maxtang、中柏等。 |
| 价格       | 16G+512G 二手 / 准新机目标 800-1100；超过 1200 要很谨慎。                    |
| 大品牌溢价 | 宏碁、零刻、机械师等不是不能买，但 N100/N150 溢价最好控制在 200 元左右。     |
| AMD 老款 U | R5 3500U 这类老低压 U 不是不能买，但功耗和发热通常高于 N100/N150；除非价格明显低或需要更强多线程/核显，否则不作为静音办公首选。 |

## 新一批里最值得追的

1. **Maxtang 大唐 Max N，N150 / 16G / 512G，1024 元**
   当前最贴合你的目标。重点核实购买记录、CPU 型号、SSD 健康度、是否带原装电源。

2. **磐镭 N100 / 16G / 512G，1100 元**
   可以聊，但最好压到 1000 左右。它是 N100，不是 N150，不值得溢价。

3. **1080 元那台 N150 / 16G / 512G**
   配置和价格对，但品牌写“Intel/英特尔”不等于品牌可靠。先要订单链接和机器铭牌。

## 当前决策规则

| 情况                                        | 动作                                         |
| :------------------------------------------ | :------------------------------------------- |
| N150 + 16G+512G + 二线品牌 + 900-1050       | 重点看，值得私聊。                           |
| N100 + 16G+512G + 二线品牌 + 800-1000       | 可以买，超过 1100 谨慎。                     |
| N100 + 8G                                   | 不买，除非确认可升级且总价仍低。             |
| 16G+256G                                    | 只在 800-900 以下考虑。                      |
| 不带电源 / 后配电源 / 有电流声 / 不议价不退 | 降低优先级，通常跳过。                       |
| 品牌写 Intel/其他/不明                      | 必须要订单截图、商品链接、铭牌、CPU-Z 截图。 |
| AMD R5 3500U / 3700U 这类老低压 U           | 只有价格明显低于 N100/N150 同配置，或你需要更强核显/多线程时才考虑；一定要现场听风扇。 |

## 闲鱼私聊核实清单

| 要问的问题                                             | 目的                                        |
| :----------------------------------------------------- | :------------------------------------------ |
| “能发系统或 BIOS 里的 CPU 型号截图吗？”                | 确认到底是 N100、N150 还是卖家写错。        |
| “能发内存容量、硬盘容量、硬盘健康度和通电时间截图吗？” | 防止 8G/16G、256G/512G 混淆，也能排除老盘。 |
| “电源是原装吗？规格是多少 V/A？”                       | 小主机电源不匹配会导致死机、重启、啸叫。    |
| “能发购买记录或商品链接吗？”                           | 判断是否还在保，以及对比新机活动价。        |
| “有没有拆修、进水、摔碰、风扇异响、电流声？”           | 二手机核心风险。                            |
| “能录一段开机进系统和待机噪音视频吗？”                 | 你的需求重心是安静办公，噪音比跑分更重要。  |
| “能录一段网页多开 / 播放视频 5 分钟后的风扇声音吗？”   | AMD 老款 U 和小机箱更要看持续负载后的噪音，而不是只看刚开机。 |
